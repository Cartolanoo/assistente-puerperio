# Configurações de Email (Opcional - se não configurar, emails serão logados no console)
# 
# OPÇÃO 1: Gmail (REQUER Verificação em Duas Etapas + Senha de App)
# ⚠️ IMPORTANTE: NÃO use a senha normal da conta Gmail!
# 1. Ative Verificação em Duas Etapas: https://myaccount.google.com/security
# 2. Gere Senha de App: https://myaccount.google.com/apppasswords
#    - Selecione "Mail" e "Outro (nome personalizado)" → "Sophia Chatbot"
#    - Copie a senha gerada (16 caracteres sem espaços)
# 3. Use essa senha no MAIL_PASSWORD (não a senha normal!)
# 4. O MAIL_USERNAME deve ser seu email Gmail completo
# 5. O MAIL_DEFAULT_SENDER será automaticamente o mesmo que MAIL_USERNAME para Gmail
MAIL_SERVER=smtp.gmail.com
MAIL_PORT=587
MAIL_USE_TLS=True
MAIL_USERNAME=seu_email@gmail.com
MAIL_PASSWORD=sua_senha_app_gerada_aqui
